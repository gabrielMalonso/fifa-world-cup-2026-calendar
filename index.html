Assinatura de verdade

Para o Google Agenda acompanhar futuras atualizações, você precisa assinar a URL do calendário. Baixar o arquivo sozinho não basta.

https://gabrielmalonso.github.io/fifa-world-cup-2026-calendar/output/world_cup_2026_fixtures.ics
  1. No Google Agenda, abra Adicionar calendário > Por URL.
  2. Cole a URL acima.
  3. Pronto: quando este mesmo link for atualizado, o Google tende a puxar as mudanças depois.
Fonte oficial API pública da FIFA descoberta pela própria página de fixtures.
Atualização O arquivo .ics pode ser regenerado e republicado no mesmo URL.
Uso no Google Agenda Importar copia. Assinar por URL acompanha mudanças depois.
